客户端使用手册
==================
.. note:: 功夫交易系统的操作界面借助 **模块化自由集成** 解决方案获得灵活性和可视性，力求更加便捷直观
  
  系统以功能为组，并分别适配不同面板，用户可根据操作习惯灵活地使用和调整面板，

  模块化系统具有更加 **灵活的统计能力**，如持仓面板可根据选中内容的不同，对策略、账户、交易算法的持仓进行分别统计。
